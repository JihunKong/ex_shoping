# 🛒 슬기로운 소비 생활

학생들에게 현명한 소비 습관을 가르치기 위한 교육용 쇼핑 시뮬레이션 앱입니다.

## 주요 기능

- 🎯 **미션 시스템**: 다양한 소비 관련 미션 수행
- - 🛍️ **장바구니**: 가상 쇼핑 장바구니 기능
  - - 💰 **예산 관리**: 예산 내 쇼핑 연습
    - - 🤖 **AI 피드백**: 소비 패턴에 대한 AI 분석 및 피드백
      - - 📊 **총액 계산**: 실시간 장바구니 금액 계산
       
        - ## 기술 스택
       
        - - **Frontend**: Streamlit
          - - **AI**: OpenAI GPT
           
            - ## 설치 및 실행
           
            - ```bash
              pip install -r requirements.txt
              streamlit run app.py
              ```

              ## 교육 목표

              - 합리적인 소비 습관 형성
              - - 예산 계획 및 관리 능력 향상
                - - 충동구매 vs 계획구매 이해
